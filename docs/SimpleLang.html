staticSimpleLang.instructionsstring

All instructions supported by the SimpleLang machine.
Properties:
Name Type Default Description
LOAD string 001 Loads the operand into the accumalator.
STORE string 010 Stores the accumalator value in memory.
ADD string 011 Adds the operand value to the accumalator.
SUB string 100 Subtracts the operand value from the accumalator.
EQUAL string 101 Jumps the next instruction iff the accumalator value equals the operand.
JUMP string 110 Jumps to the instruction specefied by the operand.
HALT string 111 Halts this machine.

accumalatornumber

The current value of the machines accumalator.

is_haltedboolean

Boolean stating if the machine is halted.

memoryArray.<number>

The current memory of the machine.

programArray.<SimpleLang.ProgramLine>

The tokenised program this machine runs.

program_counternumber

The current value of the program counter.
